## ✅ 已完成（依 spec 的 20 個階段）

### 基礎
- [x] **1. 後端骨架** — Maven、JWT、全域錯誤格式 `{code,message,details[]}`、BaseEntity(UUID+稽核)、CORS、`x-request-id`
- [x] **2. 帳號/使用者** — 註冊 / 登入(JWT+BCrypt) / `me` / 更新個人檔案
- [x] **3. 情侶綁定** — 邀請碼 → 加入 → 解除；`CoupleContext` 成員權限守門（所有 couple 資料都過它）

### 每日核心
- [x] **4. 任務 Todo** — CRUD、子任務、留言、負責人(我/對方/一起)、優先級、**驚喜任務對另一半隱形**、共同目標進度
- [x] **5. 紀念日 / 倒數** — 每年/每月重複、`daysLeft` 倒數、提醒天數
- [x] **6. 共同日曆** — 事件 CRUD、依日期區間查詢、約會額外欄位（預算/交通/穿搭…）
- [x] **7. 心情分享** — 每日打卡、emoji、需要回應、對方快速回應（抱抱你/晚點陪你…）
- [x] **8. 小紙條 / 信** — 即時或**定時解鎖未來信**（解鎖前對方看不到）、收藏、已讀
- [x] **9. 首頁聚合 API** — `GET /home`（在一起天數、對方心情、今日任務、下個紀念日、今日行程、去年今天）

### 進階
- [x] **10. 相簿 / 回憶** — 相簿、照片上傳（私有、權限控管）、「去年今天」
- [x] **11. 每日問答** — 每日一題、**雙方都答完才解鎖**彼此答案、題庫種子
- [x] **12. 願望清單** — 想一起做的事 CRUD、**可轉成任務**、完成 → 回憶
- [x] **13. 輕量記帳** — 支出記錄、分帳類型、彙總（記錄優先、不逼分帳）
- [x] **14. 約會規劃** — 候選方案 → 雙方投票 → 拍板 → **自動建立行程＋準備任務**
- [x] **15. 吵架修復** — 選狀態 → 寫感受 → AI 潤飾成溫和版 → 對方回應 → 後續任務；**安全防護**（自傷/暴力導向真人協助）
- [x] **16. AI 助手** — 任務拆解、約會建議、紀念日文案、文字潤飾、關係小提醒（可插拔：預設 stub，可接 Claude）
- [x] **17. 通知系統** — 站內通知 + 分類偏好 + FCM-ready；任務/心情/紙條自動觸發

### 前端
- [x] **18. Angular 骨架** — standalone + signals + lazy routes、JWT 攔截器、守衛、暖色(奶油白+暖橘)行動優先主題
- [x] **19. 五大頁面** — 登入/註冊/配對、底部分頁(首頁/任務/日曆/回憶/我們)＋ 我們頁的 10 個子功能
- [x] **20. README + 端對端驗證** — 後端 curl、前端瀏覽器實測
